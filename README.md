(1) 從一個數字的陣列中，如：[1,2,3,5,6,7]，任取兩個數字湊一組進行加總 (在此例中有15組要比對)，請找出是否有總和相等的組別。

請輸出「需要比對的組別數目」，並列舉總和相等的「和」，及其「組合」。
如果有多種相同總和的狀況，必須列出全部解，並依照總和的大小依序列出。
若無相同總和的解，請輸出 NO 即可。
為了容易實作，我們限定數字陣列不會超過50個數字，數字都是正整數，數字總和不會超過 10000。
假設你的程式邏輯是在 solve(int[] num) 函數內執行。測試案例之 input 已在 num 參數中。
Test Case 1
Input：
    1,2,3,5,6,7
Output:
    15 pairs
    7==>(1,6)(2,5)
    8==>(1,7)(2,6)(3,5)
    9==>(2,7)(3,6)
Test Case 2
Input：
    1,3,8,9,12
Output:
    10 pairs 
    NO
Test Case 3
Input:
    22,33,44,55,66
Output:
    10 pairs
    77==>(22,55)(33,44)
    88==>(22,66)(33,55)
    99==>(33,66)(44,55)
